# 📦 Arşiv Otomasyonu Uygulaması

NYP Arşiv Otomasyonu, dosyaları organize etmek, sıkıştırmak ve güvenli bir şekilde arşivlemek için geliştirilen C# tabanlı bir uygulamadır.

## ✨ Özellikler

- 📁 Dosya yönetimi ve organizasyonu
- 🗜️ Dosya sıkıştırma ve arşivleme
- 🔒 Güvenli şifreleme (BouncyCastle)
- 🎯 Otomatik dosya işleme
- 📊 Arşiv yönetimi ve izleme

## 🛠️ Teknoloji Yığını

- **Dil**: C#
- **Framework**: .NET Framework
- **Kriptografi**: BouncyCastle.Cryptography
- **Lisans**: MIT

## 🚀 Kurulum

1. Repository'yi klonlayın:
```bash
git clone https://github.com/bulbulsena/Archive-_Automation_Nyp.git
```

2. Proje dizinine gidin:
```bash
cd Archive-_Automation_Nyp
```

3. Visual Studio'da çalışmaya başlayın veya konsol ile derleyin:
```bash
dotnet build
```

## 📖 Kullanım

Uygulamayı başlatmak için:
```bash
dotnet run
```

Dosyaları arşivlemek ve sıkıştırmak için uygulamanın arayüzünü kullanın.
